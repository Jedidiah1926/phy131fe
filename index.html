📐 운동학 (Kinematics)

🔑 핵심 개념 3종

  • 위치 (Position) — 기준점 대비 좌표
  • 속도 (Velocity) — 위치의 시간 변화율 (벡터!)
  • 가속도 (Acceleration) — 속도의 시간 변화율 (벡터!)
v = Δx / Δt
a = Δv / Δt

📌 평균 vs 순간

  • 평균속도 = 총 변위 / 총 시간
  • 순간속도 = 극한(Δt→0) 의 Δx/Δt
  • 스칼라 vs 벡터: 속력(speed)은 스칼라, 속도(velocity)는 벡터

⚙️ 등가속도 운동 방정식 (4개)

vf = v₀ + at // Δx 없음
Δx = v₀t + ½at² // vf 없음
vf² = v₀² + 2aΔx // t 없음
Δx = ½(v₀+vf)t // a 없음

🎯 자유낙하

  • 가속도 = g = 9.8 m/s² (아래 방향)
  • 최고점에서 순간속도 = 0, 가속도 ≠ 0
  • 올라갈 때와 내려올 때 가속도 방향은 동일 (아래)
최고점에서 가속도는 0이 아님! 여전히 -9.8 m/s²

🗺️ 2D 운동 (투사체 운동)

2D 운동은 x, y 방향 독립적으로 분리해서 풀기!

x 방향: 등속 운동 (ax = 0)
x = v₀ₓ · t
y 방향: 자유낙하 (ay = -g)
y = v₀ᵧt - ½gt²
  • 초기 속도 분해: v₀ₓ = v₀cosθ, v₀ᵧ = v₀sinθ
  • 두 방향의 시간 t는 항상 공유

🔄 기준 좌표계 (Reference Frames)

같은 문제에서 모든 물체에 공통 기준계를 사용해야 함. 기준계 선택이 계산 난이도를 결정한다.

⚡ 힘과 뉴턴 법칙 (Forces & Newton's Laws)

📜 뉴턴 3법칙

  • 1법칙 (관성): 외력이 없으면 정지 or 등속 유지
  • 2법칙 (F=ma): ΣF = ma (알짜힘 = 질량 × 가속도)
  • 3법칙 (작용반작용): A가 B에 힘 → B가 A에 같은 크기, 반대 방향
ΣF = ma

💪 힘의 종류

  • 중력 (Gravity): Fg = mg, 아래 방향
  • 수직항력 (Normal): 표면에 수직, 크기는 상황에 따라 변함
  • 장력 (Tension): 줄 따라 작용, 질량 없는 줄에서 일정
  • 마찰력 (Friction): 표면에 평행, 운동/정지 마찰
  • 탄성력 (Spring): Hooke's Law: F = -kΔx

🧱 마찰력 (Friction)

  • 정지마찰 (Static): fs ≤ μs·N (최대값 있음)
  • 운동마찰 (Kinetic): fk = μk·N (일정)
  • μs > μk 항상 성립
  • 운동이 시작되면 static → kinetic으로 전환
fk = μk · N
fs,max = μs · N

🌀 스프링 (Hooke's Law)

  • F = -kΔx (복원력, 항상 평형점 방향)
  • k = 스프링 상수 (단위: N/m)
  • Δx = 평형점으로부터 변위
  • 음의 부호 → 변위 반대 방향으로 힘
F = -kΔx

⚖️ 평형 (Equilibrium)

정적 평형: ΣF = 0, a = 0 (정지 상태)
동적 평형: ΣF = 0, 등속 운동 (a = 0)

🔗 연결된 물체 & 도르래 (Pulleys)

  • 연결된 물체는 가속도가 동일 (가속도 제약 조건)
  • 도르래: 줄의 장력은 도르래 양쪽에서 같음 (마찰 없는 경우)
  • 3법칙 쌍은 서로 다른 FBD에 나타남
// Atwood Machine 예시
a = (m₁ - m₂)g / (m₁ + m₂)
💥 운동량과 충격량 (Momentum & Impulse)

📦 운동량 (Momentum)

  • 운동의 "양"을 나타내는 벡터
  • p = mv (단위: kg·m/s)
  • 벡터이므로 방향 중요!
p = mv

⚡ 충격량 (Impulse)

  • 충격량 = 운동량의 변화
  • J = ΔP = F·Δt
  • 뉴턴 2법칙을 다시 쓴 것!
J = Δp = F·Δt
// J = ∫F dt (일반형)

🔒 운동량 보존

  • 계에 외력이 없으면 총 운동량 보존
  • ΣF_ext = 0 → p_총 = 일정
  • 충돌, 폭발 문제에 핵심!
p₁ᵢ + p₂ᵢ = p₁f + p₂f

💣 충돌 3종류

  • 완전 비탄성: 합쳐짐, KE 최대 손실
    m₁v₁ = (m₁+m₂)vf
  • 비탄성: 운동량 보존, KE 일부 손실
  • 탄성: 운동량 + KE 둘 다 보존
  • 폭발: 처음 정지 → 분리, 총 운동량 = 0

🧭 2D 운동량

운동량은 벡터 → x, y 방향으로 각각 보존!

x: p₁ₓᵢ + p₂ₓᵢ = p₁ₓf + p₂ₓf
y: p₁ᵧᵢ + p₂ᵧᵢ = p₁ᵧf + p₂ᵧf
🔋 에너지와 일 (Work & Energy)

🎯 일 (Work)

  • W = F·d·cosθ (θ = 힘과 변위 각도)
  • 힘과 변위가 같은 방향 → 양의 일
  • 힘과 변위가 반대 방향 → 음의 일
  • 수직이면 → 일 = 0
W = F·d·cosθ = F⃗·d⃗

🏃 운동 에너지 (KE)

  • 운동하는 물체가 가진 에너지
  • K = ½mv²
  • 일-에너지 정리: W_net = ΔK
K = ½mv²
W_net = Kf - Ki

⛰️ 퍼텐셜 에너지 (PE)

  • 중력: Ug = mgh (h = 높이)
  • 탄성: Us = ½kx²
  • 기준점(h=0) 직접 선택 가능
  • 내부 힘이 한 일 = -ΔPE
Ug = mgh
Us = ½kx²

🔋 역학적 에너지 보존

  • 마찰 없을 때: E_tot = K + U = 일정
  • Ki + Ui = Kf + Uf
  • 마찰 있으면: 열에너지(Eth) 발생
Ki + Ui = Kf + Uf
// 마찰 없는 경우

⚡ 일률 (Power)

P = W/t = F·v

단위: W (와트) = J/s

에너지 다이어그램: 안정 평형 = 극솟값, 불안정 평형 = 극댓값

📊 에너지 원칙 (Energy Principle)

계 내부에서 에너지 변환은 일어나지만 총 에너지는 변하지 않음. 외부에서 일이 들어와야 계의 에너지가 바뀜.

ΔE_system = W_external + Q
🌀 회전 운동 (Rotation & Angular Motion)

📐 각운동 기본량

  • 각위치 θ (rad)
  • 각속도 ω = dθ/dt (rad/s)
  • 각가속도 α = dω/dt (rad/s²)
v = rω   a = rα   (선형↔각도 변환)

🔩 토크 (Torque)

  • 회전의 "원인" (힘의 회전 버전)
  • τ = r × F = rF·sinθ
  • r = 회전축으로부터 거리
  • 방향: 오른손 법칙
τ = rF sinθ
Στ = Iα

🏋️ 관성 모멘트 (Moment of Inertia)

  • 회전 저항의 척도 (질량의 회전 버전)
  • I = Σmr² (이산적)
  • I = ∫r² dm (연속)
  • 중심 기준: 막대 I = (1/12)ML², 원반 I = (1/2)MR²
I = Σmᵢrᵢ²

📏 평행축 정리

  • I = I_cm + Md²
  • d = 질량중심에서 새 축까지 거리
  • 새 축 기준의 I를 쉽게 계산
I = I_cm + Md²

🔄 구르기 (Rolling Without Slipping)

  • v_cm = Rω (구름 조건)
  • KE_total = ½mv² + ½Iω²
  • = 병진 KE + 회전 KE
v = Rω
K = ½mv² + ½Iω²

🌊 각운동량 (Angular Momentum)

  • L = Iω (회전하는 물체)
  • L = r × p (점 질량)
  • 외부 토크 = 0 → 각운동량 보존
L = Iω
Στ = dL/dt

✖️ 외적 (Cross Product)

  • 벡터 방향: 오른손 법칙
  • |A×B| = AB sinθ
  • 토크, 각운동량 계산에 사용
τ⃗ = r⃗ × F⃗

🌀 등가속 각운동 방정식

ωf = ω₀ + αt
θ = ω₀t + ½αt²
ωf² = ω₀² + 2αθ

선형 운동 방정식과 완전히 대응!

〰️ 진동과 파동 (Oscillations & Waves)

🌱 단순 조화 운동 (SHM)

  • 선형 복원력 → F = -kx
  • x(t) = A·cos(ωt + φ)
  • ω = 각진동수 (rad/s)
  • A = 진폭, φ = 초기 위상
x(t) = A cos(ωt + φ)
ω = √(k/m)   // 스프링

⏱️ 주기와 진동수

  • T = 주기 (한 번 진동하는 시간, 초)
  • f = 진동수 (Hz = 1/s)
  • ω = 각진동수 (rad/s)
T = 2π/ω = 1/f
// 스프링: T = 2π√(m/k)
// 단진자: T = 2π√(L/g)

🔭 단진자 (Simple Pendulum)

  • 복원력: F = -mg sinθ ≈ -mgθ (소각 근사)
  • 소각 근사: sinθ ≈ θ (θ << 1 rad)
  • T는 질량과 무관, 길이에만 의존
T = 2π√(L/g)

⚡ SHM 에너지

  • E_total = ½kA² = 일정
  • K와 U가 계속 변환되며 합은 일정
  • x=0에서 v=최대, U=0
  • x=±A에서 v=0, U=최대
E = ½mv² + ½kx² = ½kA²

🌊 파동 (Waves)

📏 파동의 기본 성질

  • 파장 λ: 한 주기의 공간 거리
  • 진동수 f: 초당 진동 횟수
  • 진폭 A: 최대 변위
  • 파속 v = fλ
v = fλ

🔊 세기 (Intensity)

  • I = P/A (단위면적당 파워)
  • I ∝ A² (진폭 제곱에 비례)
  • 구형파: 거리 r 증가 → I ∝ 1/r²

🚑 도플러 효과

  • 관측자/소스가 움직이면 진동수 변화
  • 가까워지면 → f 증가
  • 멀어지면 → f 감소
f_obs = f_s · (v±v_obs)/(v∓v_src)
🌡️ 열역학 (Thermodynamics)

🌡️ 온도 단위

  • 섭씨(°C): 물 0°C 얼음, 100°C 끓음
  • 화씨(°F): 32°F, 212°F
  • 켈빈(K) = °C + 273.15 (절대 0도 기준)
  • 열역학은 항상 켈빈 사용!

🤝 열역학 0법칙

  • A↔B 열평형, B↔C 열평형 → A↔C 열평형
  • 온도계의 이론적 기반
열평형 상태에서 두 물체의 온도는 같다

📏 열팽창

  • 선팽창: ΔL = αL₀ΔT
  • 부피팽창: ΔV = βV₀ΔT
  • β ≈ 3α (3D = 1D × 3)
  • α = 재료에 따른 상수
ΔL = αL₀ΔT

🔥 열과 열용량

  • Q = mcΔT (온도 변화 시)
  • Q = mL (상전이 시, ΔT=0)
  • c = 비열, L = 잠열
  • 1 kcal = 4186 J
Q = mcΔT
Q = mL // 상전이

🫧 이상기체 법칙

  • 고온, 저밀도에서 모든 기체가 따름
  • pV = NkBT (분자 수 N)
  • pV = nRT (몰 수 n)
  • kB = 1.38×10⁻²³ J/K (볼츠만 상수)
  • R = 8.314 J/(mol·K) (기체 상수)
pV = NkBT = nRT

⚗️ 기체 분자 운동론

  • 평균 KE = (3/2)kBT
  • 내부 에너지: E_int = (3/2)NkBT = (3/2)nRT
  • 온도 ↑ → 분자 속도 ↑
⟨KE⟩ = ³⁄₂ kBT
E_int = ³⁄₂ NkBT

🔁 열역학 1법칙

  • ΔE_int = Q - W
  • Q: 계로 들어온 열 (+), 나간 열 (-)
  • W: 계가 한 일 (+), 받은 일 (-)
  • 내부 에너지는 경로 무관 (상태함수)
ΔE_int = Q - W

📊 열역학 과정

  • 등온 (Isothermal): ΔT = 0 → E_int 일정
  • 단열 (Adiabatic): Q = 0 → pVᵞ = const
  • 등압 (Isobaric): Δp = 0
  • 등적 (Isochoric): ΔV = 0 → W = 0

🔥 열역학 2법칙

  • 열은 자발적으로 뜨거운 곳→ 차가운 곳
  • 엔트로피는 자발적으로 증가
  • 완벽한 효율의 열기관은 불가능
어떤 열기관도 카르노 효율보다 클 수 없다

⚙️ 열기관 & 카르노

  • 효율: η = W_out / Q_H = 1 - Q_C/Q_H
  • 카르노 효율: η_Carnot = 1 - T_C/T_H
  • T는 반드시 켈빈!
  • 이상적 가역 기관 = 카르노
η = 1 - Q_C/Q_H
η_Carnot = 1 - T_C/T_H

🌬️ 열전달 방식

  • 전도 (Conduction): 접촉, P = kA·ΔT/d
  • 대류 (Convection): 유체 흐름
  • 복사 (Radiation): 전자기파
